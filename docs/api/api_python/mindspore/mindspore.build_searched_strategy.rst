mindspore.build_searched_strategy
=======================================

.. py:class:: mindspore.build_searched_strategy(strategy_filename)

    构建网络中每个参数的策略，用于分布式推理。关于它的使用细节，请参考： `保存和加载模型（HyBrid Parallel模式） <https://www.mindspore.cn/tutorials/experts/zh-CN/r1.7/parallel/save_load.html>`_。

    **参数：**

    **strategy_filename** (str) - 策略文件的名称。

    **返回：**

    Dict，key为参数名，value为该参数的切片策略。

    **异常：**

    - **ValueError** - 策略文件不正确。
    - **TypeError** - `strategy_filename` 不是str。
